System Development Life Cycle

Utilized a Predictive development life cycle where the model is designed, implemented, and tested incrementally (piece by piece) until the product is finished - or until all requirements have been fulfilled. This model combines the elements of the waterfall model with the iterative philosophy of prototyping. In this particular model, each module passes through the requirements, design, build, and testing phases. It also involves delivering system components as they’re completed, it allows for partial utilization of product - so in that case it means completed features can be used before the full project is done - and avoids a long development time frame.
